# 💳 Simulador de Crédito Pro V15

Simulador de crédito interactivo con modo oscuro, tabla de amortización, comparador de plazos y captura de imagen.

## 🌐 Ver en línea
👉 [https://TU-USUARIO.github.io/simulador-credito](https://TU-USUARIO.github.io/simulador-credito)

## ✨ Características
- Cálculo de pagos mensuales con IVA
- Comparador de plazos con gráfica de barras
- Tabla de amortización
- Modo crédito por días
- Modo oscuro / claro
- Modo cliente (oculta controles del asesor)
- Captura de imagen de la simulación
- Compartir por WhatsApp
- Código QR con enlace

## 🚀 Uso local
Simplemente abre `index.html` en tu navegador. No requiere instalación.
